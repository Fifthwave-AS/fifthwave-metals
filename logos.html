Logo Concepts – "Flowsheet"

Four distinct logo concepts capturing the idea of a metallurgical flowsheet. Each shown on light and dark backgrounds.

Concept A – Flow Nodes

FLOWSHEET PATH
Concept: Five interconnected nodes trace a branching process path — a direct visual metaphor for a metallurgical flowsheet. The branching path at node 2 represents decision points (e.g., leach vs. concentrate routes).

Concept B – Cascading Layers

PROCESS STAGES
Concept: Three horizontal layers connected by downward lines — representing the cascading stages of a process flowsheet (Input → Processing → Output). The layout mirrors how engineers actually read flowsheets: top-to-bottom, left-to-right.

Concept C – Convergence Mark

INTEGRATED SYSTEM
Concept: Five flowing streams converge into a single output — capturing both the "five waves" brand name and how a flowsheet integrates multiple inputs (geology, chemistry, physics, data, operations) into one optimized decision. The progressive opacity represents increasing confidence.

Concept D – Circuit Block

ENGINEERING DIAGRAM
Concept: A literal simplified flowsheet circuit — process boxes connected by flow lines with a branching split and a recycle loop (dashed line). This is instantly recognizable to any mining engineer. The most technically authentic of the four concepts.
